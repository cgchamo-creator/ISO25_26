# UT01. Arquitecturas de Sistemas Operativos

Esta unidad establece los cimientos teóricos del módulo. No se puede administrar lo que no se comprende desde su diseño.

### Objetivos de la Unidad (RA1)
*   Identificar los componentes críticos de un sistema operativo.
*   Diferenciar entre tipos de núcleos (monolíticos, microkernels, híbridos).
*   Comprender la gestión de memoria y el aislamiento de procesos.

### Hoja de Ruta para el Desarrollo
1.  **Teoría**: Arquitectura de niveles, gestión de interrupciones y llamadas al sistema (Syscalls).
2.  **Tutorial**: Identificación de versiones de kernel en Linux y Windows, y análisis de la estructura de capas del sistema.
