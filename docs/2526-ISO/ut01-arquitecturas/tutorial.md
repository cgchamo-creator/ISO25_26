## Resumen del Laboratorio

En este primer tutorial práctico, tomaremos el control de la línea de comandos para ver qué está pasando "bajo el capó" de nuestros sistemas.

### Objetivos Prácticos:
*   **Identificación del Kernel**: Comprobar versiones y tipos de núcleo en Ubuntu y Windows.
*   **Gestión de Procesos**: Uso de `top`, `htop` y `Get-Process` para analizar la carga del sistema.
*   **Análisis de Memoria**: Ver cómo el SO asigna y libera recursos en tiempo real.
*   **Diagramas de flujo**: Crearemos nuestro primer diagrama Mermaid para representar el ciclo de vida de un proceso.

:::info Prerrequisitos
Para este tutorial necesitaremos una terminal Bash (Linux) y una PowerShell (Windows). No hace falta instalar software adicional por ahora.
:::
