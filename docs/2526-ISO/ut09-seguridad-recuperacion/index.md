# UT09. Seguridad y Recuperación

La última línea de defensa: auditoría, copias de seguridad y restauración.

### Objetivos de la Unidad
*   Configurar políticas de auditoría de accesos.
*   Diseñar y ejecutar planes de copias de seguridad.
*   Recuperar el sistema ante fallos críticos o desastres.

### Hoja de Ruta para el Desarrollo
1.  **Teoría**: Estrategias de backup (3-2-1), tipos de copia (incremental, diferencial), auditoría de archivos.
2.  **Tutorial**: Automatización de backups en Linux con `rsync` y en Windows Server Backup.
