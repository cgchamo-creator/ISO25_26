# UT08. Monitorización y Optimización

Mantener el sistema sano mediante la supervisión constante de sus constantes vitales.

### Objetivos de la Unidad (RA8)
*   Analizar el visor de eventos y logs del sistema.
*   Utilizar contadores de rendimiento para identificar cuellos de botella.
*   Optimizar el uso de CPU, RAM y E/S de disco.

### Hoja de Ruta para el Desarrollo
1.  **Teoría**: Logs de sistema (syslog, journalctl), visor de eventos de Windows y monitores de rendimiento.
2.  **Tutorial**: Creación de alertas de rendimiento y análisis de registros para resolución de incidencias.
