# UT05. Gestión de Procesos y Recursos

Cómo el sistema operativo maneja la ejecución de programas y servicios de fondo.

### Objetivos de la Unidad (RA5)
*   Controlar el ciclo de vida de los procesos y servicios.
*   Gestionar prioridades y estados de ejecución.
*   Automatizar tareas recurrentes del sistema.

### Hoja de Ruta para el Desarrollo
1.  **Teoría**: Estados del proceso, señales de terminación, servicios (systemd) y tareas programadas.
2.  **Tutorial**: Gestión de demonios en Linux y servicios en Windows. Uso de Crontab y Programador de tareas.
