## Resumen del Laboratorio

Pondremos en práctica la planificación de la unidad anterior. Utilizaremos entornos virtuales para simular servidores reales.

### Objetivos Prácticos:
*   **Preparación de la VM**: Configuración de BIOS/UEFI, RAM y almacenamiento.
*   **Particionado Manual**: Crear estructuras personalizadas en el instalador de Ubuntu.
*   **Instalación de Windows Server**: Selección de versión (Standard/Datacenter) y experiencia de escritorio vs Core.
*   **Post-Instalación**: Instalación de Guest Additions/VM Tools e identificación de drivers básicos.

:::info Herramientas
Utilizaremos VirtualBox o VMware Player para estos laboratorios. Asegúrate de tener las imágenes ISO descargadas de las fuentes oficiales.
:::
