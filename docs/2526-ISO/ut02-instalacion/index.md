# UT02. Instalación de Sistemas Operativos

Aprenderemos a realizar despliegues profesionales, analizando el hardware y eligiendo el esquema de particionado óptimo.

### Objetivos de la Unidad (RA2)
*   Planificar instalaciones basadas en requisitos técnicos.
*   Dominar el particionado moderno (GPT) y el arranque UEFI.
*   Instalar y configurar software de base en entornos libres y propietarios.

### Hoja de Ruta para el Desarrollo
1.  **Teoría**: Comparativa de sistemas de archivos, tablas de particiones (MBR vs GPT) y modos de instalación.
2.  **Tutorial**: Instalación paso a paso de Windows Server 2022 y Ubuntu Server con particionado manual.
