# UT06. Configuración de Red

Conectar nuestros sistemas al mundo y asegurar que la comunicación sea fluida.

### Objetivos de la Unidad (RA6)
*   Configurar parámetros TCP/IP en diferentes sistemas operativos.
*   Realizar resolución de nombres local (archivo hosts).
*   Diagnosticar problemas de conectividad básicos.

### Hoja de Ruta para el Desarrollo
1.  **Teoría**: Pilas de protocolos, direccionamiento estático vs dinámico (DHCP) y comandos de diagnóstico.
2.  **Tutorial**: Configuración de red mediante Netplan (Ubuntu) y PowerShell (Windows Server).
