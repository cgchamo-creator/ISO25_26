# UT04. Gestión de Usuarios y Grupos

Administración de la identidad y el acceso en servidores y estaciones de trabajo.

### Objetivos de la Unidad (RA4)
*   Crear y gestionar cuentas de usuario y grupos de seguridad.
*   Administrar perfiles de usuario y entornos de escritorio.
*   Automatizar la gestión de cuentas mediante scripts.

### Hoja de Ruta para el Desarrollo
1.  **Teoría**: Tipos de cuentas, grupos locales, perfiles móviles y políticas de contraseñas.
2.  **Tutorial**: Scripting para la creación masiva de usuarios en Linux (useradd) y Windows (New-LocalUser).
