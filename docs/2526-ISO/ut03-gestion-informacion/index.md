# UT03. Gestión de la Información

El corazón de la administración: cómo se guardan los datos y quién puede acceder a ellos.

### Objetivos de la Unidad (RA3)
*   Gestionar estructuras de directorios y archivos.
*   Configurar atributos y permisos básicos.
*   Implementar seguridad avanzada mediante listas de control de acceso (ACLs).

### Hoja de Ruta para el Desarrollo
1.  **Teoría**: Estructura del árbol de directorios en Linux (FHS) y Windows. Permisos especiales (SUID, SGID, Sticky Bit).
2.  **Tutorial**: Laboratorio de permisos cruzados y gestión de ACLs desde PowerShell y Bash.
